# EuroBusinessHub

AI Business Operating System for Europe.

## Structure

| Folder | Purpose |
|--------|---------|
| `frontend/` | React + Vite + TypeScript client |
| `backend/` | Future API services (placeholder) |
| `database/` | Schema, migrations, seeds (placeholder) |
| `shared/` | Cross-package types and contracts |
| `docs/` | Architecture and product documentation |

## Quick start

```bash
cd frontend
npm install
npm run dev
```

## Status

Foundation phase — mock data only, no external API connections.
